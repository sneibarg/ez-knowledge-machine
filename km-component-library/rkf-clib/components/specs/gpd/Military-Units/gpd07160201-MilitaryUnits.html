Assets

SHAKEN supports all "asset" classes supported in nuSketch:

  • Anti-Tank-Unit
  • Armor-Unit
  • Artillery-Unit
  • Aviation-Unit
  • Cavalry-Unit
    • Armored-Cavalry-Unit
    • Light-Cavalry-Unit
  • Engineer-Unit
  • Infantry-Unit
    • Light-Infantry-Unit
    • Mechanized-Infantry-Unit
  • Main-Command-Post-Unit
  • Rear-Command-Post-Unit
  • Special-Forces-Unit
  • Tactical-Command-Post-Unit
  • Task-Organized-Unit

Echelons

SHAKEN supports the following echelons. A Unit at a given echelon can contain Units at any of the echelons below it.

Corps
Division
Brigade
Group / Regiment
Battalion / Squadron
Company / Battery / Troop
Platoon / Detachment
Section
Squad
Crew / Team

A Military Unit is normally defined by combining one or more assets with an echelon (for example, Artillery-Battery, Light-Infantry-Brigade).
